# Manufacturing KPI Dashboard

## Overview

Python project developed to analyze manufacturing performance and monitor key production indicators.

## KPIs

- Production Volume
- Yield
- Scrap Rate
- Downtime
- Defects by Machine

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

python generate_data.py
