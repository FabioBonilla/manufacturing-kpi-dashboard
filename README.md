## Overview

The Manufacturing KPI Dashboard is a data analytics project designed to monitor and visualize key manufacturing performance indicators.

The dashboard provides operational visibility into production efficiency, quality performance, downtime events, and process effectiveness through interactive visualizations and KPI tracking.

This project demonstrates how manufacturing data can be transformed into actionable insights that support operational excellence and continuous improvement initiatives.

## Key Performance Indicators

### Production Volume

Tracks total units produced during a specific period.

### Yield

```text
Yield = Good Units / Total Units Produced
```

Measures the percentage of products meeting quality standards.

### Scrap Rate

```text
Scrap Rate = Scrap Units / Total Units Produced
```

Measures material and production losses.

### Downtime

Tracks machine stoppages and production interruptions.

### Defects by Machine

Analyzes quality issues associated with manufacturing equipment.
